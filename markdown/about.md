All pages under [kevinrsx.github.io](https://kevinrsx.github.io) are created with

- [easy pandoc templates](https://github.com/ryangrose/easy-pandoc-templates/blob/master/html/uikit.html)

Convert markdown to html:

```
pandoc <markdown> -o <html> --template=uikit.html --toc
```

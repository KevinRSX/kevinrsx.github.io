All pages under [kevinrsx.github.io](https://kevinrsx.github.io) are created with

- [pandoc](https://pandoc.org/)
- [Bootstrap](https://getbootstrap.com/docs/4.0/getting-started/introduction/), and
- [uikit](https://gist.github.com/KevinRSX/19d935d20026bc9b2d8fec663b8a73c5) (modification of [easy-pandoc-templates](https://github.com/ryangrose/easy-pandoc-templates/blob/master/html/uikit.html))

Convert X (markdown, Latex, etc.) to html:

```
# default template dir: ~/.pandoc/template
pandoc /path/to/X -o /path/to/html --template=uikit.html --toc
```
